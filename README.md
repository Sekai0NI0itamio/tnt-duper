# TNT Duper

## Endless Explosive Automation

Turn dispensers into infinite TNT factories! Simply place a TNT block into any dispenser, power it, and watch it spawn primed TNT entities **without consuming the original block**. Perfect for rapid excavation, mob farms, or chaotic contraptions.

### Key Features

- **Infinite TNT Dispensing**: Every activation spawns a new TNT entity while keeping the original block intact.
- **Vanilla-Compatible**: Works with all standard redstone triggers—buttons, pressure plates, and observer clocks.
- **Instant Activation**: Spawned TNT is already primed, exploding 4 seconds after placement.
- **Multi-Dispenser Support**: Scale up your explosions by loading multiple dispensers in a single system.

No configuration needed—just add TNT to any dispenser and let the chaos begin!
